# Crawl Customs Data

## 抓取海关统计数据在线查询平台数据

## Usage

    ```
    按照商品编码、年月日检索所需数据
    检索过程中存在滑块验证码过程，中途使用js脚本进行绕过
    最后将将获得到的商品数据存在本地json（需改进写入mongo）
    ```

## 返回结果的数据结构

```json
  {
  "id": "10011900",
  "product_name": "其他硬粒小麦",
  "first_q": "501",
  "first_n": "加拿大",
  "second_q": "10",
  "second_n": "一般贸易",
  "rmb": "11"
},
```
